    Catatan K-Means Clustering

    Pengertian K-mean#

    K-Means itu salah satu algoritma unsupervised learning buat ngelompokkin data berdasarkan kemiripan. Jadi misalnya kita punya data acak, K-Means bakal ngelompokin data itu ke dalam beberapa cluster, berdasarkan jarak terdekat antar titik dan pusat kelompok (centroid).

    Tujuan

    Membagi data ke dalam K kelompok supaya data yang mirip-mirip masuk ke kelompok yang sama.

    Istilah Penting

    • Cluster: Kelompok data yang punya ciri atau pola yang mirip.
    • Centroid: Titik pusat dari suatu cluster.
    • Euclidean Distance: Cara ngukur jarak antar titik.

    Langkah-Langkah K-Means

    1. Tentukan nilai K (jumlah cluster).
    2. Pilih K centroid awal (biasanya acak).
    3. Hitung jarak tiap data ke semua centroid (pakai rumus Euclidean).
    4. Kelompokin data ke centroid terdekat.
    5. Update posisi centroid berdasarkan rata-rata data dalam cluster.
    6. Ulangi langkah 3–5 sampai konvergen.

    Rumus Euclidean Distance

    d = √((x₁ - c₁)² + (x₂ - c₂)²)

    Kelebihan K-Means

    • Simpel dan cepat
    • Cocok buat data besar
    • Mudah diimplementasikan

    Kekurangan K-Means

    • Harus tentuin K di awal
    • Hasil bisa beda tergantung centroid awal
    • Kurang cocok kalau bentuk cluster-nya aneh (non-bulat)

    Cara Menentukan K

    • Elbow Method: Lihat grafik error vs K, cari "titik siku".
    • Silhouette Score: Nilai antar -1 s/d 1. Semakin tinggi semakin bagus.

    Menghitung K-Mean

    import pandas as pd
from sklearn.datasets import load_iris
from sklearn.cluster import KMeans
from sklearn.metrics import silhouette_score
import matplotlib.pyplot as plt

# Load dataset iris dari sklearn
iris = load_iris()
X = pd.DataFrame(iris.data, columns=['sepal_length', 'sepal_width', 'petal_length', 'petal_width'])

# Ubah ke array NumPy (jika diperlukan)
X_values = X[['petal_length', 'petal_width', 'sepal_length', 'sepal_width']].values

# Jalankan KMeans dengan 4 cluster
kmeans = KMeans(n_clusters=4, random_state=42, n_init=10)
kmeans.fit(X_values)
labels = kmeans.labels_

# Hitung SSE dan Silhouette Score
sse = kmeans.inertia_
silhouette = silhouette_score(X_values, labels)

print("SSE:", sse)
print("Silhouette Score:", silhouette)

# Visualisasi hasil cluster (gunakan 2 fitur: petal_length dan petal_width)
plt.figure(figsize=(8, 6))
plt.scatter(X_values[:, 0], X_values[:, 1], c=labels, cmap='viridis', s=50)
plt.scatter(kmeans.cluster_centers_[:, 0], kmeans.cluster_centers_[:, 1],
            s=100, c='red', marker='X', label='Centroids')
plt.xlabel('Petal Length')
plt.ylabel('Petal Width')
plt.title('Hasil K-Means Clustering (4 Cluster)')
plt.legend()
plt.grid(True)
plt.tight_layout()
plt.show()
