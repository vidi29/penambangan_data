    Local Outlier Factor (LOF) – Deteksi Data Pencilan

    Dalam analisis data, kita sering menemukan anomali atau outlier, yaitu data yang berbeda jauh dari data lain dalam kumpulan data. Misalnya, dalam data transaksi keuangan, anomali bisa menjadi indikasi penipuan. Salah satu metode yang sering digunakan untuk mendeteksi outlier adalah Local Outlier Factor (LOF).

    LOF adalah metode berbasis kepadatan lokal yang membandingkan kepadatan suatu titik data dengan tetangganya. Jika kepadatan titik tersebut jauh lebih kecil dibandingkan tetangga-tetangganya, maka titik tersebut dianggap sebagai outlier.

    Konsep Dasar LOF

    LOF bekerja dengan membandingkan kepadatan lokal suatu titik terhadap kepadatan lokal tetangganya. Langkah-langkah utama dalam LOF adalah sebagai berikut:

    1. jarak Tetangga Terdekat (k-NN Distance)
      2. Untuk setiap titik data, hitung jarak ke k tetangga terdekat. Biasanya digunakan jarak Euclidean:

      Contoh: jika kita punya dua titik data A (2.1, 3.2) dan B (1.8, 2.9), maka:

    2. Jarak Keterjangkauan (Reachability Distance)
      3. Jarak keterjangkauan antara dua titik adalah jarak maksimum antara jarak aktual dan jarak k-tetangga.

      Jika jarak aktual A ke B lebih kecil dari jarak k-NN B, maka gunakan jarak k-NN B.
    3. Kepadatan Lokal (Local Density)
      4. Kepadatan lokal dihitung sebagai kebalikan dari rata-rata jarak keterjangkauan ke tetangga-tetangga terdekatnya:

      Semakin kecil LD, semakin jarang data tersebut muncul.
    4. Faktor LOF
      5. Faktor LOF diperoleh dengan membandingkan kepadatan lokal titik tersebut dengan kepadatan lokal tetangganya:

      Jika LOF > 1, titik lebih jarang muncul dibandingkan tetangganya (indikasi outlier). Jika LOF ≈ 1, maka titik memiliki kepadatan yang mirip dengan tetangganya (bukan outlier).

    Contoh Implementasi LOF Menggunakan Library Sklearn

    Data Awal

    data = np.array([
    [2.1, 3.2], [1.8, 2.9], [2.5, 3.5],
    [8.0, 8.2], [8.1, 8.0], [1.9, 3.1],
    [2.3, 3.3], [8.2, 8.1], [7.9, 8.3], [2.0, 3.0]
])

df = pd.DataFrame(data, columns=["X1", "X2"])

    Model LOF dan Hasil

    # Inisialisasi model LOF dengan k=3
lof = LocalOutlierFactor(n_neighbors=3, metric="euclidean")

# Fit model dan dapatkan skor LOF (semakin negatif, semakin outlier)
lof_scores = lof.fit_predict(data)
lof_values = -lof.negative_outlier_factor_

# Buat DataFrame untuk melihat hasilnya
df = pd.DataFrame(data, columns=["X1", "X2"])
df.insert(0, "ID", range(1, 11))  # Menambahkan kolom ID dari 1-10
df["LOF"] = lof_values
df["Outlier"] = ["Ya" if x > 1.2 else "Tidak" for x in lof_values]

# Tampilkan hasil
print(df)

    Visualisasi Outlier dengan LOF

    # Visualisasi data
plt.figure(figsize=(8, 6))
sns.scatterplot(x=df["X1"], y=df["X2"], hue=df["Outlier"], palette={"Ya": "red", "Tidak": "blue"}, s=100)
plt.title("Deteksi Outlier dengan LOF")
plt.xlabel("X1")
plt.ylabel("X2")
plt.show()
